Scene_GenieW trakcie

Zaawansowany plugin Unreal Engine 5 wykorzystujący sztuczną inteligencję do automatycznego generowania scen 3D

Przegląd

Scene_Genie to przełomowy plugin dla Unreal Engine 5, który integruje najnowsze modele sztucznej inteligencji do inteligentnego generowania i aranżacji scen 3D. Plugin wykorzystuje zaawansowane algorytmy AI do analizowania właściwości zasobów, kompozycji sceny i intencji użytkownika, umożliwiając tworzenie przekonujących środowisk 3D za pomocą naturalnych poleceń językowych.

Kluczowe funkcje

Wsparcie wielu LLM

Claude 4, GPT-4.1, Gemini 2.5 Pro, OpenAI O3

Analiza wizualna

Automatyczna ekstrakacja właściwości wizualnych i semantycznych

Kontekst sceny

Analiza kompozycji, oświetlenia i środowiska

Wykonywanie działań

Automatyczne rozmieszczanie, rotacja i stylizacja

Technologie

  • Unreal Engine 5.6+ - Platforma bazowa z natywną integracją
  • C++ - Podstawowy język programowania dla komponentów core
  • Python - Skrypty analizy danych i przetwarzania
  • Multi-LLM API - Integracja z Claude, GPT, Gemini
  • JSON Schema - Walidacja odpowiedzi AI
  • Native UI - Interfejs użytkownika UE5 z drag-drop

Wyzwania i rozwiązania

Integracja wielu modeli AI

Stworzenie zunifikowanego interfejsu dla różnych dostawców LLM z różnymi formatami API i możliwościami. Rozwiązanie obejmuje abstrakcję warstwy komunikacji i adaptacyjne formatowanie zapytań.

Analiza kontekstu sceny

Implementacja kompleksowej analizy sceny 3D obejmującej kompozycję, oświetlenie, gęstość obiektów i wydajność. System automatycznie klasyfikuje środowisko i dostosowuje strategie rozmieszczania.

Wykonywanie działań w czasie rzeczywistym

Zaprojektowanie systemu wykonującego polecenia AI bezpośrednio w edytorze UE5 z walidacją błędów i feedbackiem. Obsługa 9 różnych typów akcji z bezpiecznym wykonywaniem i możliwością cofania.

Zakres projektu

Funkcjonalności obecne

  • Interaktywne okno pluginu z intuicyjnym interfejsem
  • Drag-drop zasobów z przeglądarki zawartości
  • Natywne miniatury zasobów z metadanymi
  • Wymuszanie schematu JSON w odpowiedziach LLM
  • Wykonywanie funkcji wysyłanych przez LLM
  • Integracja z viewport i analiza wizualna

Planowane ulepszenia

  • Przeprojektowanie promptu opisu sceny
  • Dodanie szczegółów rotacji globalnej zasobów
  • Rozszerzenie systemu analizy performansy
  • Wsparcie dla procedural generation

Przyszły rozwój

Projekt rozwija się w kierunku bardziej zaawansowanych funkcji AI, w tym uczenia maszynowego do optymalizacji rozmieszczenia, integracji z systemami procedural generation i rozszerzenia na inne silniki gier. Planowane jest również dodanie wsparcia dla collaborative editing i cloud-based scene generation.
